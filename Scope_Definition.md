# Scope Definition

## Objective

Create a chat agent for **SquareTrade** that appears as a pop-up on the
website and provides automated assistance for buying plans, claims, and
support topics using official SquareTrade resources.

------------------------------------------------------------------------

## In-Scope Features

### 1. UI Integration

-   Pop-up chat widget on SquareTrade website.

### 2. Knowledge Base

-   Use only official content from:
    -   SquareTrade Get Help
    -   SquareTrade Support
    -   Info about Protection Plans
-   No external or unrelated answers.

### 3. Core Capabilities

-   Answer FAQs about:
    -   Buying protection plans.
    -   Filing and tracking claims.
    -   Coverage details.
-   Provide step-by-step guidance for common tasks.

### 4. Escalation

-   If intent is unmatched or user requests human help:
    -   Escalate to live agent via chat or callback.

### 5. Compliance

-   No personal data storage.

------------------------------------------------------------------------

## Out-of-Scope

-   Answering questions unrelated to SquareTrade plans or claims.
-   Providing financial advice or third-party product support.
-   Handling payments directly in chat.
